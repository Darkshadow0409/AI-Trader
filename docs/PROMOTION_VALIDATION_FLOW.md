# Promotion And Validation Flow

Strategy lifecycle states:

- experimental
- paper_validating
- promoted
- demoted

Promotion inputs:

- backtest robustness
- walk-forward quality
- forward paper-validation results
- sample size
- data quality and realism penalties

Demotion triggers:

- degraded forward results
- realism penalties crossing promotion thresholds
- weak walk-forward or calibration support relative to the current promoted state

Calibration remains a bucket comparison tool. It is not a probability-of-profit claim.
