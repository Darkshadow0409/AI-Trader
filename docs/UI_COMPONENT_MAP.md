# UI Component Map

## Shell

- `App.tsx`: shell orchestration, tab switching, focus state, and refresh fan-out
- `TopRibbon.tsx`: global macro, freshness, risk budget, pipeline, and gate ribbon
- `LeftRail.tsx`: persistent navigation plus watchlist and scout queue
- `ContextSidebar.tsx`: right-side context stack
- `CommandCenter.tsx`: safe operational actions and runtime visibility

## Core workspace blocks

- `PriceChart.tsx`: dense focus chart
- `SignalDetailsCard.tsx`: evidence, data reality, catalyst, targets, invalidation, and linked risk
- `SignalTable.tsx`: sortable and filterable signal workbench

## Major operator views

- `DeskTab.tsx`: what-matters-now summary
- `TradeTicketsTab.tsx`: ticket lifecycle, checklist, shadow mode, reconciliation
- `ActiveTradesTab.tsx`: lifecycle management, timeline, stress, and execution realism
- `JournalTab.tsx`: review workflow and decision-quality analytics
- `SessionDashboardTab.tsx`: review cadence, daily briefing, and backlog
- `StrategyLabTab.tsx`: promotion, validation, calibration, and realism penalties
- `ReplayTab.tsx`: replay frames, event windows, and stress
- `PilotDashboardTab.tsx`: gate state, divergence, adapter health, and audit events

## Shared visual building blocks

- `Panel.tsx`
- `StateBlock.tsx`
- `HeatmapGrid.tsx`
- `EquityCurveChart.tsx`

These components are intentionally plain and high-density rather than brand- or marketing-oriented.
